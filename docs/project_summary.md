## 权限设计

系统预设了5个角色，满足您的需求：

1. **管理员（admin）**：所有权限
2. **采购（purchaser）**：可以创建BOM、报价单、合同
3. **审批人（approver）**：可以审批/拒绝报价单
4. **财务（finance）**：可以创建/查看合同
5. **普通用户（user）**：只能查看

关键权限点：
- ✅ 合同录入需要 `contract:create` 权限
- ✅ 报价审批需要 `quotation:approve` 权限
- ✅ 报价拒绝需要 `quotation:reject` 权限

## 开发建议

1. **开发顺序**：建议按照优先级逐步开发，先保证核心业务流程可用
2. **测试**：每完成一个模块，建议进行充分测试，特别是Excel导入功能
3. **数据导入**：历史数据导入时，建议先在小批量数据上测试校验逻辑
4. **权限测试**：开发过程中注意测试不同角色的权限控制

## 快速开始

详细的环境搭建步骤请参考：`docs/getting_started.md`

简要步骤：
1. 创建MySQL数据库
2. 配置`.env`文件
3. 安装Python依赖
4. 运行初始化脚本：`python app/core/init_db.py`
5. 启动后端服务：`uvicorn app.main:app --reload`
