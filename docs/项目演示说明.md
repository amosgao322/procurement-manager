## 系统界面预览（功能说明）

### 1. 登录页面
- 输入用户名和密码登录
- 默认管理员账户：admin / admin123（首次登录后需修改密码）

### 2. 主界面布局
- **顶部导航栏**：显示当前用户、退出登录
- **左侧菜单**：根据用户权限显示可访问的功能模块
- **主内容区**：显示各功能页面

### 3. BOM清单管理页面

**功能入口**：左侧菜单 → BOM管理

**主要功能**：
- **BOM列表**：显示所有BOM清单，支持搜索、筛选
- **创建BOM**：点击"新建"按钮，填写BOM基本信息，添加明细项
- **编辑BOM**：点击列表中的BOM，可以修改信息和明细
- **导入BOM**：点击"导入"按钮，上传Excel文件
  - 系统自动解析Excel
  - 显示导入结果和错误提示
  - 错误信息精确到行号和字段
- **导出BOM**：将BOM导出为Excel文件
- **删除BOM**：删除不需要的BOM（需确认）

**BOM明细编辑**：
- 可以逐行添加物料
- 支持复制、粘贴
- 自动计算总价（数量×单价）

### 4. 供应商管理页面

**功能入口**：左侧菜单 → 供应商管理

**主要功能**：
- **供应商列表**：显示所有供应商，支持搜索
- **新建供应商**：填写供应商基本信息
- **编辑供应商**：修改供应商信息
- **导入供应商**：批量导入供应商数据
- **查看详情**：查看供应商的报价历史、合同历史

### 5. 报价单管理页面

**功能入口**：左侧菜单 → 报价管理

**主要功能**：
- **报价单列表**：显示所有报价单，可按状态筛选（草稿、待审批、已审批、已拒绝）
- **创建报价单**：
  - 选择供应商
  - 选择关联的BOM（可选）
  - 填写报价明细
  - 填写报价日期、有效期、付款条件等
- **提交报价单**：将草稿状态的报价单提交审批
- **审批报价单**（需要审批权限）：
  - 查看报价单详情
  - 点击"审批通过"或"拒绝"
  - 填写审批意见
- **查看报价单**：查看报价单的完整信息和审批历史

**状态流转**：
```
草稿 → 提交 → 待审批 → 审批通过/拒绝
```

### 6. 合同管理页面

**功能入口**：左侧菜单 → 合同管理

**主要功能**：
- **合同列表**：显示所有合同，支持搜索、筛选
- **创建合同**：
  - 选择供应商
  - 关联BOM和报价单（可选）
  - 填写合同信息
  - 上传合同文件（PDF等）
- **编辑合同**：修改合同信息
- **查看合同**：查看合同详情和关联信息

### 7. 询比价页面

**功能入口**：左侧菜单 → 询比价

**主要功能**：
- **选择BOM**：选择要询价的BOM清单
- **生成询价单**：基于BOM生成询价需求
- **比价分析**：
  - 选择多个供应商的报价单
  - 系统自动对比分析
  - 显示多维度评分结果
  - 推荐最优供应商

**比价结果展示**：
- 每个物料的供应商报价对比表
- 综合评分排名
- 价格趋势图（如有历史数据）

### 8. 操作日志页面

**功能入口**：左侧菜单 → 操作日志

**主要功能**：
- **日志列表**：显示所有操作记录
- **筛选条件**：
  - 按用户筛选
  - 按操作类型筛选（创建、修改、删除、审批等）
  - 按资源类型筛选（BOM、供应商、报价单、合同）
  - 按时间范围筛选
- **查看详情**：查看操作前后的数据对比
- **导出日志**：导出日志为Excel文件

### 9. 统计报表页面（后续开发）

**功能入口**：左侧菜单 → 统计报表

**主要功能**：
- 报价统计：按供应商、时间统计报价情况
- 合同统计：合同金额、数量统计
- 供应商统计：供应商合作情况统计

## 操作流程示例

### 示例1：创建BOM并导入报价

1. **创建BOM**
   - 进入"BOM管理"
   - 点击"新建"
   - 填写BOM名称、产品名称
   - 点击"导入明细"，上传Excel文件
   - 系统自动解析并填充明细
   - 检查错误提示，修正后保存

2. **创建报价单**
   - 进入"报价管理"
   - 点击"新建"
   - 选择供应商
   - 选择刚创建的BOM
   - 系统自动填充报价明细（可修改）
   - 填写报价日期、有效期等
   - 保存为草稿或直接提交

3. **审批报价单**
   - 审批人登录系统
   - 进入"报价管理"，筛选"待审批"
   - 查看报价单详情
   - 点击"审批通过"或"拒绝"
   - 填写审批意见

### 示例2：询比价流程

1. **选择BOM**
   - 进入"询比价"
   - 选择要询价的BOM清单

2. **查看历史报价**
   - 系统自动显示该BOM的历史报价
   - 可以查看不同供应商的历史报价

3. **比价分析**
   - 选择多个供应商的报价单
   - 点击"比价分析"
   - 查看比价结果和推荐供应商

4. **创建合同**
   - 根据比价结果选择供应商
   - 进入"合同管理"
   - 创建新合同，关联报价单

## 数据导入示例

### BOM导入Excel格式

```
序号 | 设备名称 | 规格型号 | 单位 | 数量 | 单价 | 总价 | 备注
-----|---------|---------|------|------|------|------|------
1    | 堵头    | DN15, 304不锈钢 | 套 | 9 | 2.5 | 22.5 |
2    | 手柄蝶阀 | DN50(尼龙板、QT450体) | 片 | 26 | 60 | 1560 |
```

**注意事项**：
- 设备名称和数量为必填
- 数量和价格必须是数字
- 如果总价为空，系统会自动计算（数量×单价）

### 导入错误提示示例

```
导入完成，共50行数据
成功：48行
错误：2行

错误详情：
- 第3行：数量格式错误，应为数字
- 第15行：设备名称不能为空
```
