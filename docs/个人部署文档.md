第一步:
# 构建镜像（首次构建可能需要 5-10 分钟）
docker build -t procurement-manager:latest .
(如果失败可能是镜像源地址网络问题，可以开一下vpn试试)

第二步 导出镜像
docker save procurement-manager:latest -o procurement-manager-latest.tar

第三步 上传镜像到远程服务器
scp procurement-manager-latest.tar ubuntu@114.132.198.141:/tmp/

第四步 在服务器上导入镜像
docker load < /tmp/procurement-manager-latest.tar

# 停止旧容器（如果存在）
docker stop procurement-manager
docker rm procurement-manager

第五步 启动镜像bash
docker run -d \
  --name procurement-manager \
  --restart unless-stopped \
  -p 8080:80 \
  --env-file .env \
  -v $(pwd)/backend/uploads:/app/uploads \
  -v $(pwd)/logs:/app/logs \
  procurement-manager:latest
```
